# FaceterToken

Download:
```bash
git clone https://github.com/kirillsurkov/FaceterToken.git
cd FaceterToken
```

Install dependencies:
```bash
npm install
```

Compile contracts:
```bash
truffle compile
```

Run tests:
```bash
truffle test --network test
```

Configure:
```javascript
// truffle.js
const bufferKey = ["ce2eab5......."]; // buffer private key
```

Deploy:
```bash
truffle migrate
```
